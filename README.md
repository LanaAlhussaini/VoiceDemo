# VoiceDemo 🎤

A simple voice interaction demo built using web technologies.  
This project demonstrates speech recognition and voice-based interaction directly in the browser.

## 🚀 Live Demo
Visit the project here:  
[VoiceDemo Live Site](https://lanaalhussaini.github.io/VoiceDemo/?utm_source=chatgpt.com)

## ✨ Features
- Voice input using browser speech recognition
- Real-time transcript display
- Simple and responsive UI
- Interactive voice demo experience

## 🛠️ Technologies Used
- HTML
- CSS
- Web Speech API
